Problem FD -- разряды чисел Фибоначчи

Дано целое положительное число x

Будем обозначать fib(x) = число Фибоначчи под номером x

fib(0) = 0

fib(1) = 1

fib(90) = 2880067194370816120

Вам необходимо написать программу, которая вычисляет последний разряд в десятичной записи fib(x)

Посылка должна состоять из программы, считывающей со стандартного ввода число x и выводящей на стандартный вывод число fib(x) mod 10


Пример 1:

Вход: 0

Выход: 0


Пример 2:

Вход: 1

Выход: 1


Пример 3:

Вход: 90

Выход: 0


Пример 4:

Вход: 91

Выход: 9
