Problem SCB — обобщённый бинарный поиск

Бинарный поиск можно осуществлять в массиве произвольных элементов, обладая только информацией о размере элемента и функцией-компаратором. В этом задании считайте тип компаратора заданным заранее

typedef int (*cmp_t)(const void *lhs, const void *rhs);

Компаратор возвращает отрицательное число, если lhs логически меньше, чем rhs, 0 если они равны и положительное число, если lhs логически больше, чем rhs

Ваша задача — написать функцию обобщённого бинарного поиска с заранее заданной сигнатурой

void *cbsearch(const void *key, const void *base, int num, int size, cmp_t cmp);

Здесь key это элемент, который ищется, base это массив в котором осуществляется поиск, size это размер элементов массива (и, соответственно, элемента key), num это количество элементов в массиве

Решение более чем с логарифмической сложностью вряд ли уложится в лимит по времени

Посылка должна состоять из функции cbsearch (допустимо также любое количество вспомогательных), и не должна содержать функцию main

Функция cbsearch должна возвращать указатель на найденный элемент. Если ничего не найдено то можно вернуть нулевой указатель или указатель за концом массива
