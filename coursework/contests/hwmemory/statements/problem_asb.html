Problem ASB — количество установленных бита в массиве

Ваша задача написать функцию arrpopcount с наперёд заданным прототипом, которая возвращает количество установленных бит в массиве

int arrpopcount(const unsigned char *parr, int len);

Если в массиве не установлено ни одного бита, верните 0

Посылка должна состоять из этой одной функции (допустимо также любое количество вспомогательных), и не должна содержать функции main


Пример 1

Вход: {0, 0, 0, 0, 0}, 5

Выход: 0


Пример 2

Вход: {1, 2, 4, 8, 16}, 5

Выход: 5


Пример 3

Вход: {26, 9, 1, 11, 217}, 5

Выход: 14
