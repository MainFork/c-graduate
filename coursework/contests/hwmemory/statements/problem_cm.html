Problem CM — обобщённое слияние

Нетривиальной, но интересной задачей является сортировка объектов разных размеров, последовательно расположенных в памяти

Предположим, что у вас есть реализованный кем-то компаратор

typedef int (*xcmp_t)(void *lhs, int lsz, void *rhs, int rsz);

Ваша задача реализовать функцию сортировки использованием этого компаратора. Сигнатура этой функции задана заранее

void xmsort(void *mem, int *sizes, int nelts, xcmp_t cmp);

Здесь mem это начало памяти, sizes это массив размеров

Это задание рассчитано на то, что вы реализуете сортировку слиянием, но в принципе вы не ограничены в выборе алгоритма. Просто учтите, что решение с квадратичной сложностью вряд ли пройдёт проверку.

Посылка должна состоять из функции xmsort (допустимо также любое количество вспомогательных), и не должна содержать функцию main
