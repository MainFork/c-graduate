Problem CSE — обобщённая сортировка выбором

Сортировка выбором использует линейный поиск в несортированной части массива и обмен с подходящим значением в сортированной части и её конкретной версии посвящена Problem SSE. Настало время обобщить сделанное

Итерацию сортировки выбором можно сделать, обладая только информацией о размере элемента и функцией-компаратором. В этом задании считайте тип компаратора заданным заранее

typedef int (*cmp_t)(void const * lhs, void const * rhs);

Компаратор возвращает 1 если lhs логически меньше, чем rhs, иначе 0
