Problem SI — сортировка вставками

Основная часть алгоритма сортировки вставками выглядит следующим образом

void inssort(int *arr, int len) {
  int i;
  for (i = 0; i < len; ++i) {
    int key, pos;
    key = arr[i];
    pos = moveright(arr, key, i);
    arr[pos] = key;
  }
}

Использованная здесь процедура moveright сдвигает к младшим адресам текущий ключ от позиции i к той позиции, которую он должен занять. Сама вставка на вычисленную позицию происходит после неё

Ваша задача написать эту функцию с заранее заданной сигнатурой

int moveright(int *arr, int key, int last) {
  // TODO: ваш код здесь
}

Здесь arr это указатель на первый элемент массива, key это ключ, позиция для которого ищется, last это позиция ключа (или иначе первая позиция за последним отсортированным элементом). Функция возвращает позицию вставки ключа. На этой позиции должно остаться то значение, с которого началось распространение вправо

Посылка должна состоять из этой одной функции (допустимо также любое количество вспомогательных), и не должна содержать функцию main


Пример 1

Вход: arr = {1, 2, 4, 5, 3, 2}, key = 3, last = 4

Выход: arr = {1, 2, 4, 4, 5, 2}, pos = 2


Пример 2

Вход: arr = {5, 2, 4, 1, 3, 2}, key = 5, last = 0

Выход: arr = {1, 2, 4, 4, 5, 2}, pos = 2
