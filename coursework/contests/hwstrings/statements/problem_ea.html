Problem EA — аргументы функции main

Напишите программу которая будет печатать аргументы с которыми вызвана, заменяя '-' на '+'

> ./a.out -a t -x --b 123
-a t +x ++b 123

Если среди этих аргументов есть -r, программа должна вывести их в обратном порядке

> ./a.out -a t -r -x --b 123
123 ++b +x +r t +a

Посылка должна состоять из программы, которая распечатывает результат на стандартный вывод
